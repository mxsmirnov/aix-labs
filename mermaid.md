## Диаграмма последовательности

```mermaid
sequenceDiagram
    participant U as Пользователь
    participant FE as Фронтенд
    participant A as AuthService
    participant DB as База данных
    
    U->>FE: Вводит логин/пароль
    FE->>A: POST /api/login
    A->>DB: Проверка учетных данных
    DB-->>A: Данные пользователя
    A->>A: Генерация JWT токена
    A-->>FE: 200 OK + JWT
    FE-->>U: Успешный вход
    
    Note over U,DB: Сессия установлена
    U->>FE: Запрос выписки
    FE->>A: GET /api/statement (с JWT)
    A->>A: Валидация токена
    A-->>FE: Данные выписки
    FE-->>U: Отображение выписки
```
