# aix-labs
Материалы курса AIX-Online (Проектирование ИТ-решений...)

Структура репозитория:
```
.
├── dot         # Диаграммы GraphViz (dot формат)
|   └── img     # Изображения (экспортированы плагином Graphviz (dot) language support for Visual Studio Code)
├── lg          # Простой ИИ-агент(Python) на LangGraph с DeepSeek LLM и инструментом-калькулятором
├── mcp-server  # Простейший MCP сервер(Python) для чтения локального файла через stdio
├── mermaid     # Диаграммы Mermaid
|   └── img     # Изображения (экспортированы командной строкой)
├── openai-api  # Примеры вызовов API
├── plantuml    # Диаграммы PlantUML
|   └── img     # Изображения (экспортированы командной строкой)
└── vector      # Простые примеры BoW и Word2Vec
```
